El pretérito pluscuamperfecto

había + participio pasado

¿Para qué sirve?

Se utiliza para indicar una acción pasada que ocurrió antes que otra acción también pasada. Es el "pasado del pasado".

Línea de tiempo:

"Cuando llegué a la fiesta (Pasado A), Juan ya se había ido (Pasado B - ocurrió primero)."

Formación

Se forma con el pretérito imperfecto de haber + el participio pasado.

Persona Haber (Imperfecto) Ejemplo
yohabíahabía estudiado
túhabíashabías comido
él / ella / ustedhabíahabía vivido
nosotros/ashabíamoshabíamos leído
vosotros/ashabíaishabíais visto
ellos / ellas / ustedeshabíanhabían hecho

Dato útil: Las formas de 'yo' y 'él/ella/usted' son idénticas (había).

Palabras clave

Este tiempo suele aparecer acompañado de expresiones que marcan el orden de los eventos:

Ya
(Already)
Antes
(Before)
Todavía no
(Not yet)

Práctica rápida

¿Qué había pasado antes? Haz clic para ver la respuesta:

1. Yo nunca ____ (ver) algo tan bonito.

Yo nunca había visto algo tan bonito.

2. Cuando llegamos, ellos ya ____ (salir).

Cuando llegamos, ellos ya habían salido.

3. ¿(Tú) ____ (estudiar) español antes de este año?

¿Habías estudiado español antes de este año?

4. Nos dijeron que (ustedes) ____ (ganar).

Nos dijeron que habíais ganado.
