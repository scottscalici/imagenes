Lectura JSON Generator

Generates a single items["id"] entry that matches your current renderer, plus optional fields for future upgrades.

1) Metadata

Used to build ID + URL path.

Format: lectura-s4-rosalia

Final URL will be: /lectura-sX-slug

2) Images (optional)

Will also populate legacy imagen so your current renderer keeps working.

3) Reading text

Paste text. Paragraphs split on blank lines. Lines starting with - are kept as separate bullet lines.

Parsed output (contenido.texto)

    4) Answer reveal settings

    Your renderer enforces this. Generator only writes the contenido.revelado config.

    5) Questions

    Add questions. Options can be pasted as lines; correct answers selected via radio/checkbox.

    6) Output

    Tip: If you want zero surprises, keep schema stable: imagen (legacy) + imagen_principal (new).
